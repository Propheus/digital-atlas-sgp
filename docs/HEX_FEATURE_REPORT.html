Key insight: influence features are the backbone; raw features suffer from curse of dimensionality

The 4 influence pillars (121 features) are the only ones whose removal degrades kNN. Removing influence_transit_max alone drops accuracy by 9.2 percentage points. Meanwhile, removing place_composition (66 features, the largest pillar) actually improves accuracy by +11.7%.

This does NOT mean place composition is useless — it carries the most variance (22.3%) and correctly differentiates Sungei Kadut (84% automotive) from Orchard (31% shopping). What it means is that in a raw 309-dimensional cosine-similarity space, 66 category features dilute the spatial-context signal. A learned compression (autoencoder) would resolve this by weighting features appropriately.

4.2 Nearest-Neighbor Structural Twins

Sentosa → Marina Bay Sands (the signature finding)

Sentosa (SISZ01, tourism/hospitality) finds Marina Bay Sands (DTSZ12) as its 4th nearest neighbor — a hex in a completely different planning area, 4km away, but sharing the same tourism/hospitality/entertainment function. The transit influence graph connects them via MRT. This is structural similarity across geographic boundaries — exactly what a universal representation should capture.

Bedok HDB → 7 different planning areas

Bedok North HDB (BDSZ04) finds its nearest neighbors in Woodlands, Bukit Merah, Hougang, Sengkang, Queenstown, Bukit Batok — all HDB heartlands. Population range 3,049–7,666, HDB blocks 10–22. The representation correctly identifies "HDB heartland" as a structural type that spans Singapore, not a geographic cluster.
