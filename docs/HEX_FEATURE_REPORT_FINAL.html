5. Structural Twins

Sentosa → Marina Bay Sands

Sentosa (tourism) finds MBS (Downtown Core, 4km away) as its nearest non-Sentosa neighbor. Same tourism/entertainment function across geographic boundaries — connected by the MRT transit graph.

Bedok HDB → 7 planning areas

A Bedok HDB hex finds structural twins in Woodlands, Bukit Merah, Hougang, Sengkang, Queenstown, Bukit Batok, Toa Payoh. "HDB heartland" is a structural type that spans Singapore.

Raffles Place → CBD core

1,356 places (densest hex) → Downtown Core, Orchard, Singapore River, Museum, Outram. Clean commercial core cluster.
